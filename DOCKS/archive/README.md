# DOCKS/archive/ - 開発履歴アーカイブ

## 📦 このディレクトリについて

このディレクトリには、v1.0リリース完了後の開発フェーズ用ドキュメントがアーカイブされています。

これらのドキュメントは開発履歴として保持されていますが、今後の開発では通常参照しません。
