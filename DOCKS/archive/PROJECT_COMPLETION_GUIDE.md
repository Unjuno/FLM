## ビルド・テスト実行

### ビルドコマンド

```bash
# 開発ビルド
npm run tauri dev

# プロダクションビルド
npm run tauri build
```

### テストコマンド

```bash
# 単体テスト
npm test

# 型チェック
npm run type-check

# リンター
npm run lint

# E2Eテスト
npm run test:e2e
```

### Rustビルド・チェック

```bash
# Rustコードチェック
cd src-tauri
cargo check

# Clippy（警告チェック）
cargo clippy

# テスト
cargo test
```
