FLMの機能

FLMは、誰でも簡単にローカルLLMのAPIを作成・管理できるツールです。以下の5つの主要機能で構成されています。
WEBサイトに設定できる使い捨てのAPIも簡単に作成でき、このWEBサイト自体がHTML/CSS/JSだけで動作する静的サイトです。

✨

API作成機能

FLMの核心機能です。複雑な設定やプログラミング知識がなくても、 ワンクリックでローカルLLMのAPIを作成できます。

  • モデル選択からAPI作成まで、すべてGUIで完結
  • ポート番号や認証設定を自動で最適化
  • Ollamaとの自動連携で、セットアップ手順を簡略化
  • 作成したAPIはすぐに利用可能

💡 こんな場面で使えます

チャットボットや自動応答システムを作りたい時、コード生成支援ツールを作りたい時など、 AIを使ったアプリケーションを開発する際に便利です。
特に、HTML/CSS/JSだけで動作する静的WEBサイトに設定できる使い捨てのAPIを作成したい場合に最適です。 一時的なプロジェクト、デモサイト、プロトタイプ開発など、短期間だけ必要なAPIを素早く作成・削除できます。

🎯
💬

API利用機能

作成したAPIを簡単にテスト・利用できます。ブラウザベースのチャットインターフェースで、 すぐにAPIの動作を確認できます。

  • 組み込みのチャットインターフェースで即座にテスト可能
  • サンプルコード（curl、Python、JavaScript）を自動生成
  • APIの詳細情報を一目で確認
  • エンドポイントURLとAPIキーを簡単にコピー

💡 こんな場面で使えます

作成したAPIが正しく動作するか確認したい時、他のアプリケーションでAPIを使う際の サンプルコードが必要な時などに便利です。

🔌
⚙️

API管理機能

作成したAPIを一元管理できます。起動/停止、設定変更、削除など、 すべての操作を直感的なUIで行えます。

  • 複数のAPIを一覧表示で管理
  • ワンクリックでAPIの起動・停止
  • ポート番号や認証設定の変更
  • 不要になったAPIの簡単な削除

💡 こんな場面で使えます

複数のAPIを使い分けたい時、設定を変更したい時、不要なAPIを整理したい時などに便利です。

📋
🤖

モデル管理機能

Ollamaモデルの検索、ダウンロード、管理を簡単に行えます。 用途に応じて最適なモデルを選択できます。

  • 人気モデルの検索とカテゴリー別フィルタリング
  • ワンクリックでモデルのダウンロード
  • ダウンロード進捗のリアルタイム表示
  • インストール済みモデルの一覧表示と削除

💡 こんな場面で使えます

新しいAIモデルを試してみたい時、用途に合わせて最適なモデルを選びたい時、 ストレージを節約するために不要なモデルを削除したい時などに便利です。

📦
🔒

セキュリティ機能

APIキーの生成、管理、再生成を安全に行えます。 すべてのデータはローカルで処理され、プライバシーが完全に保護されます。

  • 自動生成される強力なAPIキー
  • APIキーの一覧表示と管理
  • ワンクリックでAPIキーの再生成
  • すべてのデータ処理がローカルで完結

💡 こんな場面で使えます

APIキーが漏洩した可能性がある時、セキュリティを強化したい時、 複数のAPIキーを管理したい時などに便利です。

🛡️

🛠️ 技術的な特徴

🔐 セキュリティ

  • APIキーはAES-256-GCMで暗号化保存
  • すべてのデータ処理がローカルで完結
  • インターネットへのデータ送信なし
  • プライバシーを完全に保護

⚡ パフォーマンス

  • GPU（NVIDIA GPU）対応で高速化
  • 複数のAPIを同時実行可能
  • メモリ効率的な設計
  • リソース使用量の最適化

🔌 互換性

  • OpenAI互換APIエンドポイント
  • 標準的なHTTP/JSONプロトコル
  • curl、Python、JavaScript対応
  • 既存のアプリケーションと統合可能

🌐 WEBサイト用使い捨てAPI & 静的サイト対応

📄 使い捨てAPIとは

使い捨てAPIは、一時的なプロジェクトやデモサイト、プロトタイプ開発などで 短期間だけ必要なAPIを素早く作成・削除できる機能です。

  • 数分でAPIを作成・削除可能
  • 設定が簡単で、すぐに利用開始
  • 不要になったら簡単に削除
  • ローカルで動作するため安全

🌍 静的サイト対応

このWEBサイト自体がHTML/CSS/JSだけで動作する静的サイトです。 サーバーサイドの処理が不要で、どこでもホスティング可能です。

  • HTML/CSS/JSのみで動作
  • GitHub Pages、Netlify、Vercel対応
  • サーバーサイド処理不要
  • 高速で安全な配信

💡 使い捨てAPIの活用例

🎨 デモサイト

クライアントへのデモ用に一時的なAPIを作成。 デモ終了後はすぐに削除できます。

🧪 プロトタイプ開発

アイデアの検証用に素早くAPIを作成。 検証後は不要になったら削除。

📚 学習・教育

学習用の一時的なAPIを作成。 学習終了後は簡単に削除できます。

💻 システム要件

最小要件

  • OS: Windows 10/11、macOS 11.0+、Ubuntu 20.04+
  • RAM: 8GB
  • ストレージ: 10GB（モデル含む）
  • CPU: 64-bit プロセッサ

推奨要件

  • RAM: 16GB以上
  • ストレージ: 20GB以上
  • GPU: NVIDIA GPU（オプション、高速化のため）
  • インターネット接続: 初回セットアップ時のみ必要
