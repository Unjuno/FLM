プライバシーポリシー

最終更新日:

1. はじめに

FLM（Local LLM API Manager）は、お客様のプライバシーを尊重します。 このプライバシーポリシーは、当ウェブサイトおよびアプリケーションがどのように個人情報を収集、使用、保護するかを説明します。

2. 収集する情報

2.1 ウェブサイトでの情報収集

当ウェブサイトでは、以下の情報を収集する場合があります：

  • ダウンロード統計情報（オプトイン方式）
    • OS情報（Windows、macOS、Linuxなど）
    • ダウンロード日時
    • この情報はローカルストレージに保存され、外部サーバーには送信されません
  • ブラウザ情報
    • OS検出のためのUser-Agent情報（ローカルでのみ使用）

重要: ダウンロード統計の収集は、ユーザーの明示的な同意がある場合のみ行われます。 同意がない場合、統計情報は収集されません。

2.2 アプリケーションでの情報収集

本アプリケーションは、すべてのデータをユーザーのローカルデバイスに保存します。 以下の情報がローカルに保存される場合があります：

  • API設定情報（エンドポイント、認証情報など）
  • リクエストログとパフォーマンスメトリクス
  • アプリケーション設定（テーマ、言語設定など）
  • モデル管理情報
  • 監査ログ（IPアドレスはハッシュ化、ユーザーエージェントは簡略化）

3. データの使用目的

収集したデータは、以下の目的でのみ使用されます：

  • ダウンロード統計の分析（ウェブサイト改善のため）
  • APIリクエストの処理とログ記録
  • パフォーマンス監視と分析
  • アプリケーション機能の提供
  • ユーザー設定の保存と復元

4. データの保護

本アプリケーションは、以下の方法でデータを保護します：

  • APIキーなどの機密情報の暗号化保存（AES-256-GCM）
  • OAuthトークンの暗号化保存
  • ローカルデータベースへの安全な保存
  • IPアドレスのハッシュ化（監査ログ）
  • ユーザーエージェントの簡略化（監査ログ）
  • ネットワーク通信のセキュリティ（HTTPS使用時）

5. 外部サービスへの送信

本アプリケーションは、ユーザーの許可なく、個人情報を外部サービスに送信しません。 すべてのデータ処理は、ユーザーのローカルデバイス上で行われます。

以下の場合にのみ、外部サービスと通信します：

  • GitHub Releases API（最新バージョン情報の取得）
  • Hugging Face API（モデル検索、ユーザーが明示的に実行した場合のみ）
  • リモート同期機能（ユーザーが明示的に有効化した場合のみ）

6. ダウンロード追跡の管理

当ウェブサイトでは、ダウンロード統計の収集をオプトイン方式で行っています。

6.1 同意の取得

初回ダウンロード時に、統計収集への同意を求めるダイアログが表示されます。 ユーザーは「同意する」または「拒否する」を選択できます。

6.2 設定の変更

同意設定は、ブラウザのローカルストレージに保存されます。 設定を変更するには、ブラウザの開発者ツールから以下のキーを削除または変更してください：

  • flm_tracking_consent: 'true'（同意）または 'false'（拒否）
  • flm_downloads: ダウンロード履歴（削除すると履歴が消去されます）

7. Cookieとローカルストレージ

当ウェブサイトでは、Cookieを使用しません。 ただし、以下の目的でローカルストレージを使用します：

  • ダウンロード統計の保存（ユーザーの同意がある場合のみ）
  • プライバシー設定の保存
