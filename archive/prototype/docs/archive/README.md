# アーカイブディレクトリ

このディレクトリには、プロジェクトの開発過程で作成された古いドキュメントやレポートが保存されています。

## 📁 ディレクトリ構成

### `static-analysis/`
静的解析レポートのアーカイブです。開発過程で作成された複数の静的解析レポートが含まれています。

**内容:**
- 静的解析レポート（複数バージョン）
- 機能別解析レポート
- 問題箇所一覧
- コンパイルエラー解析

**注意:** これらのレポートは開発履歴として保持されていますが、最新の情報についてはプロジェクトのメインドキュメントを参照してください。

### `release/`
リリース準備過程で作成されたドキュメントのアーカイブです。

**内容:**
- リリース準備ステータスレポート（複数バージョン）
- リリース完了報告（複数バージョン）
- リリース準備サマリー（複数バージョン）

**注意:** 最新のリリース情報については、以下を参照してください：
- `docs/release/FINAL_RELEASE_STATUS.md` - 最新のリリースステータス
- `docs/release/RELEASE_CHECKLIST.md` - リリースチェックリスト
- `RELEASE_NOTES.md` - リリースノート
- `CHANGELOG.md` - 変更履歴

## 📄 ルートディレクトリのアーカイブファイル

以下のファイルもアーカイブされています：

- `FIX_PLAN.md` - プロジェクト修正計画
- `FIX_SUMMARY.md` - 修正実施サマリー

これらのファイルは開発過程で作成されたもので、現在は参考情報として保持されています。
