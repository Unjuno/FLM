## GitHub Pages公開手順

### 4.1 リポジトリ準備

```bash
# リポジトリ作成
git init
git remote add origin https://github.com/[username]/localai-hub-website.git

# ファイル追加
git add .
git commit -m "Initial commit: website files"
git push -u origin main
```

### 4.2 GitHub Pages有効化

1. GitHubリポジトリの「Settings」へ移動
2. 「Pages」セクションを開く
3. 「Source」でブランチ選択（`main`）
4. 「/ (root)」または「/docs」を選択
5. 「Save」をクリック

**公開URL**:
```
https://[username].github.io/localai-hub-website/
```

### 4.3 カスタムドメイン設定（オプション）

1. リポジトリルートに `CNAME` ファイル作成
2. 内容: `flm.com`（取得したドメイン名）
3. DNS設定:
   - Aレコード: `185.199.108.153`, `185.199.109.153`, `185.199.110.153`, `185.199.111.153`
   - CNAMEレコード: `[username].github.io`

### 4.4 自動デプロイ設定（GitHub Actions）

`.github/workflows/deploy.yml`:
```yaml
name: Deploy to GitHub Pages

on:
  push:
    branches:
      - main

jobs:
  deploy:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - name: Deploy
        uses: peaceiris/actions-gh-pages@v3
        with:
          github_token: ${{ secrets.GITHUB_TOKEN }}
          publish_dir: ./
```

## Netlify公開手順

### 5.1 Netlifyアカウント作成

1. https://www.netlify.com/ にアクセス
2. GitHubアカウントでサインアップ（推奨）
3. アカウント作成完了

### 5.2 サイト作成

1. Netlifyダッシュボードで「Add new site」
2. 「Import an existing project」を選択
3. GitHubリポジトリを選択
4. ビルド設定:
   - Build command: （不要、静的サイトのため）
   - Publish directory: `/` または `/dist`
5. 「Deploy site」をクリック

**公開URL**:
```
https://[random-name].netlify.app/
```

### 5.3 カスタムドメイン設定

1. サイト設定 → 「Domain management」
2. 「Add custom domain」をクリック
3. ドメイン名入力
4. DNS設定（Netlifyが案内）:
   - AレコードまたはCNAMEレコード
5. SSL証明書自動取得（Let's Encrypt）

### 5.4 自動デプロイ設定

- GitHub連携で自動デプロイ（デフォルト）
- プルリクエスト毎にプレビュー作成
- 本番デプロイは `main` ブランチへpush時

## Vercel公開手順

### 6.1 Vercelアカウント作成

1. https://vercel.com/ にアクセス
2. GitHubアカウントでサインアップ（推奨）
3. アカウント作成完了

### 6.2 プロジェクト作成

1. Vercelダッシュボードで「Add New Project」
2. GitHubリポジトリを選択
3. プロジェクト設定:
   - Framework Preset: Other（静的サイト）
   - Build Command: （不要）
   - Output Directory: `/` または `/dist`
4. 「Deploy」をクリック

**公開URL**:
```
https://[project-name].vercel.app/
```

### 6.3 カスタムドメイン設定

1. プロジェクト設定 → 「Domains」
2. ドメイン名入力
3. DNS設定（Vercelが案内）
4. SSL証明書自動取得
