## Steam準備

### 1.1 Steamworksアカウント作成

1. https://partner.steamgames.com/ にアクセス
2. Steamアカウントでログイン（既存アカウントまたは新規作成）
3. Steamworksパートナー登録申請
4. 必要情報入力:
   - 開発者名
   - 連絡先情報
   - 支払い情報（Steam Direct登録料 $100 が必要）

### 1.2 Steam Direct登録

**費用**: $100（1タイトルあたり、返金不可）

**登録手順**:
1. Steamworksパートナーダッシュボードにアクセス
2. 「Steam Direct」セクションへ移動
3. 登録料支払い（$100）
4. 基本情報入力

**注意事項**:
- 登録料は1タイトルあたり
- 返金不可
- 一度登録すれば、そのアカウントで複数タイトルを公開可能

### 1.3 必要資料の準備

**必須資料**:
- [ ] アプリ名: "FLM"
- [ ] アプリID: Steamが自動生成
- [ ] 開発者名・パブリッシャー名
- [ ] サポート連絡先（メールアドレス）
- [ ] ストアページ用の画像・動画
- [ ] 説明文（英語・日本語）
- [ ] システム要件
- [ ] スクリーンショット（5-10枚）

## ストアページ作成

### 3.1 基本情報

**ストアページURL**: 
```
https://store.steam.com/app/[APP_ID]/LocalAI_Hub/
```

**タイトル**:
```
FLM - Create Local LLM APIs with One Click
```

**短い説明** (Short Description):
```
Create and use local LLM APIs without coding. One-click API creation, built-in authentication, cross-platform support.
```

**長い説明** (About This App):

```markdown
## FLM

**Create local LLM APIs with one click - no coding required.**

FLM makes it easy for anyone to create and use local LLM APIs, without technical knowledge or programming skills.

## Features

✅ **One-Click API Creation**
   - Select a model → Click "Create" → Get a secure API endpoint
   - No coding, no configuration files, no command line

✅ **Built-in Security**
   - Automatic API key generation
   - Authentication layer built-in
   - Secure by default

✅ **Cross-Platform**

   - Same great experience on all platforms

✅ **OpenAI Compatible**
   - Works with any OpenAI-compatible client
   - Standard REST API endpoints

✅ **Privacy First**
   - All data stays local
   - No cloud dependency
   - Your models, your data, your control

## Use Cases

- Non-developers who want to add AI features
- Developers who want privacy-focused LLM APIs
- Anyone who wants to avoid cloud API costs
- Local-first AI applications

## How It Works

1. Install FLM
2. Install Ollama (one-time setup)
3. Select a model
4. Click "Create API"
5. Start using your secure API endpoint

## Technical Details

- Built with Tauri (cross-platform desktop framework)
- Uses Ollama for LLM execution
- Express.js for authentication layer
- SQLite for local storage
- OSS-first approach

## Requirements

- Ollama installed (download from ollama.ai)
- 8GB+ RAM (16GB recommended)
- 10GB+ storage space
- GPU optional (CPU works too)

- Website: [URL]
- Support: [Email]

**Note**: This is the MVP version. Full cross-platform support and advanced features coming in v1.0.
```

## ビルド・アップロード手順

### 4.1 Steam用ビルド作成

#### Windowsビルド

```bash
# Steam用ビルド設定
npm run tauri build -- --target x86_64-pc-windows-msvc

# 生成されるファイル:
# src-tauri/target/release/localai-hub.exe
# src-tauri/target/release/bundle/msi/localai-hub_1.0.0_x64_en-US.msi
```

**Steam用に必要な設定**:
- `tauri.conf.json` でSteam対応設定
- Steamworks SDK統合（オプション、アチーブメント等で使用する場合）

#### macOSビルド（v1.0以降）

```bash
# Intel Mac
npm run tauri build -- --target x86_64-apple-darwin

# Apple Silicon
npm run tauri build -- --target aarch64-apple-darwin
```

#### Linuxビルド（v1.0以降）

```bash
npm run tauri build -- --target x86_64-unknown-linux-gnu
```

### 4.2 Steamworks SDK統合（オプション）

**使用する場合**:
- アチーブメント機能
- Steamクラウド保存
- Steam統計

**統合手順**:
1. Steamworks SDKダウンロード
2. Rust用バインディング追加（Tauriの場合）
3. 機能実装

**注意**: MVPではSteamworks SDK統合は不要（最小限の実装）

### 4.3 デポット作成・アップロード

1. Steamworksダッシュボードで「デポット」作成
2. Steam Pipeを使用してビルドアップロード

```bash
# Steam Pipe インストール（Steamworks SDKに含まれる）
# ビルドアップロードコマンド
steamcmd +login [username] [password] +run_app_build [app_build_script] +quit
```

**アップロードファイル**:
- インストーラー（.msi / .dmg / .deb 等）
- 必要に応じて追加ファイル（DLL等）

### 4.4 ビルド設定

**Steamworks Builder設定**:
- デフォルトブランチ: `main` または `release`
- ビルド出力パス
- インストールディレクトリ

## 料金・収益

### 7.1 Steam Direct登録料

- **初回**: $100（返金不可）
- **2タイトル目以降**: $100/タイトル

### 7.2 売上分配

**Steam手数料**:
- 売上の30%（標準）
- $10M超の場合、段階的に減額（最大25%）

**開発者への支払い**:
- 月次で支払い（最小$100から）
- 税務書類必要

### 7.3 価格戦略

**推奨価格**:
- **無料**: 最大のアクセス数確保、フィードバック収集
- **$4.99-9.99**: 適度な障壁、収益化
- **$19.99以上**: プレミアムポジション

**FLM推奨**: 
- MVP: 無料（フィードバック収集）
- v1.0以降: $4.99-9.99を検討
