## [1.0.0] - 2024年

### 🎉 初回リリース

FLM（Local LLM API Management Tool）の最初の安定版リリースです。

### ✨ 新機能

#### API管理機能
- **F001: API作成機能**
  - モデル選択からAPI作成まで、直感的なUIで操作可能
  - 自動でAPIキー生成と暗号化保存
  - Ollama自動起動・確認機能
  - 認証プロキシ自動起動

- **F002: API利用機能**
  - 作成済みAPIのテスト用チャットインターフェース
  - OpenAI互換APIとして利用可能
  - エンドポイント情報とAPIキーの表示
  - サンプルコード（curl、Python、JavaScript）の生成

- **F003: API管理機能**
  - APIの起動/停止管理
  - 設定変更（ポート番号、認証設定）
  - API削除と関連リソースのクリーンアップ
  - APIキー再生成機能

#### モデル管理機能
- **F004: モデル管理機能**
  - Ollamaモデルの検索・フィルタ・ソート
  - モデルダウンロード（進捗表示）
  - インストール済みモデル一覧
  - モデル情報の詳細表示
  - モデル削除機能

#### 認証機能
- **F005: 認証機能**
  - APIキー生成（32文字以上、ランダム）
  - APIキーの暗号化保存（AES-256-GCM）
  - Bearer Token認証
  - APIキー再生成・削除機能

#### Ollama自動インストール
- **F009: Ollama自動インストール機能**
  - システムパス上のOllama検出
  - ポータブル版Ollamaの自動ダウンロード
  - GitHub Releases APIからの自動ダウンロード
  - 自動起動と状態監視
  - エラーハンドリング（非開発者向けメッセージ）

#### 公式Webサイト
- **F008: 公式Webサイト**
  - ホームページ（プロダクト紹介）
  - ダウンロードページ（OS自動検出）
  - 機能紹介ページ
  - 使い方ガイドページ
  - FAQページ
  - お問い合わせページ
  - レスポンシブデザイン（モバイル対応）
  - アクセシビリティ対応（WCAG 2.1 AA準拠）
  - GitHub Pages自動デプロイ

### 🔧 技術仕様

- **UIフレームワーク**: Tauri 2.x
- **フロントエンド**: React 19.x + TypeScript
- **バックエンド**: Rust
- **データベース**: SQLite
- **LLM実行**: Ollama（ユーザー事前インストール）
- **認証プロキシ**: Express.js + express-http-proxy

### 📊 パフォーマンス最適化

- フロントエンド: useCallback、useMemoによる関数・計算結果のメモ化
- バックエンド: データベースインデックス（10個）、準備済みステートメント、Repositoryパターン
- ストリーム処理のメモリ最適化（バッファサイズ制限10KB、メモリリーク防止処理）
